Human Safety Cost — Jhuman

Drag the joint sliders to pose the planar robot. The cost function evaluates all 14 control points (2 per link) against each human sphere and sums the maximum exponential proximity cost per sphere.

Cost breakdown — Jhuman
Jhuman = wh · Σi maxj[exp(−α·ReLU(dij))]
—
ReLU(dij) = max(dij, 0)
Inside sphere: d < 0 → ReLU = 0 → exp(0) = 1.0 (saturated)
At surface: d = 0 → cost = 1.0
Outside: d > 0 → exp(−αd) ∈ (0,1), decays smoothly
maxj over 14 CPs → worst link drives cost
Σi over all spheres → no body segment ignored
dij(q) = ‖xj(q) − pi‖₂ − ri   →   signed proximity: positive outside, zero at surface, negative inside sphere
Jhuman(xt) = wh · Σi=1Nh maxj[ exp(−α · ReLU(dij(q))) ]
Devised for visualisation of human safety cost.
